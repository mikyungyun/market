# Market

Serverless Carrot Market Clone using NextJS, Tailwind, Prisma, PlanetScale and Cloudflare.

**PlanetScale Database**

```
Create DB
pscale database create [name] --region [area]

Connect DB
pscale connect [name]
```

**Prisma Client**

```
클라이언트 생성
npx prisma generate

Prisma 관리자 실행
npx prisma studio
```
